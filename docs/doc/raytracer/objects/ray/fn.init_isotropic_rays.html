pub fn init_isotropic_rays(start_x: f32, start_y: f32) -> Vec<ObjectRay>
Expand description

Creates a collection of rays arranged in an isotropic (point source) pattern.

This function generates rays that emanate from a central point in all directions, similar to a point light source. The rays are evenly distributed around 360 degrees.

§Arguments

  • start_x - X coordinate of the emitter’s center point
  • start_y - Y coordinate of the emitter’s center point

§Returns

A vector of ObjectRays arranged in a circular pattern from the given point
